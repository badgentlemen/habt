# habt
## CLI для поиска статей на habr.com по фразе 

---
## Установка и запуск

1. `git clone https://github.com/badgentlemen/habt.git`
2. `yarn install` для установки зависимостей
3. `yarn start` для запуска проект

---
## Стек технологий
* NodeJS 11+
* ES6 / async await / Promise
* Typescript
* yarn
